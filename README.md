# WatSys Organization Configuration

This repository contains the public GitHub organization profile and shared
community standards for WatSys Enterprises.

## Contents

- `profile/README.md` — Public organization profile
- `CONTRIBUTING.md` — Shared contribution guidelines
- `CODE_OF_CONDUCT.md` — Community behavior standards
- `SECURITY.md` — Security reporting instructions
- `SUPPORT.md` — Support guidance
- `.github/ISSUE_TEMPLATE/` — Default issue templates
- `.github/PULL_REQUEST_TEMPLATE.md` — Default pull request template

## Visibility

This repository is public because GitHub requires the organization-level
community health repository to be public for most defaults to apply.

Do not store confidential company information in this repository.
